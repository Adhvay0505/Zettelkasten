Active Directory (AD) is a directory service developed by Microsoft that manages and stores information about networked resources (users, computers, printers, etc.)
in a domain-based structure. It is used for centralized authentication and authorization.

---

### ðŸ“Œ Key Concepts

- **Directory Service:** Stores information in a hierarchical structure.
- **Objects:** Entities like users, groups, computers, etc.
- **Domain:** Logical grouping of objects that share a common AD database.
- **LDAP (Lightweight Directory Access Protocol):** Protocol used to query and modify AD.
- **Kerberos:** Default authentication protocol used in AD.
- **NTLM:** Older authentication protocol still used in some cases.
